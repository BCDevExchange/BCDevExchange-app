BC Developers' Exchange

An experiment in tech innovation and collaboration

The Idea

Create an ecosystem of innovation and collaboration between tech entrepreneurs and British Columbia's public sector.

How?

  • Share British Columbia's public sector code, APIs and data with tech entrepreneurs.
  • Collaborate in new ways with tech entrepreneurs to develop public sector code, APIs and technology products and services.
  • Build better digital services for the public sector and build business.

The Experiment

BCDevExchange is an experiment; an idea that needs to be proven. We will test, validate and refine the idea with tech entrepreneurs and early adopters in the public sector.

Make Stuff

The BCDevExchange experiment is about innovation and collaboration. One way to innovate is to share public sector data, code and APIs with tech entrepreneurs. Below is a list of data and APIs you can use right now for your business.

Data

BC Data Catalogue

Discover data under permissive licenses such as the Open Government License - British Columbia.

APIs

BC Laws API

BC Laws is an electronic library providing free public access to the laws of British Columbia made available for use under the permissive Queen's Printer License.

TransLink Open API

The TransLink Open API provides access to data associated with the regional bus network, bus service operations and major road network allowing developers to build applications enriched with TransLink data.
